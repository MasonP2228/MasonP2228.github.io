Machine Learning for the Benefit of Human Learning

For my Data Analytics minor, I was tasked with completing a data mining project on a topic of my choice. I chose to base my project around a dataset on student performance on math questions. Using Python, I implemented data mining and machine learning techniques such as logistic regression, K-nearest neighbors, and a decision tree algorithm. At the end of the project, I wrote a report and created a poster to showcase my findings.
