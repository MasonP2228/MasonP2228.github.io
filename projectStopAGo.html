Stop-A-Go

Buckle up, and get ready to go!
Stop-A-Go is an interactive game that provides a classic arcade experience. Watch the lights go round and round, and press the button when the green panel is lit up to score! If you mistime your press, however, you will get a game over.
Stop-A-Go was made using an Arduino microcontroller, which controls numerous components, including a strip of LEDs, two buttons (one button to start the game and another to play it), a sound chip, and an LCD screen that displays players' scores. The game operates using C++ computer code written in the Arduino IDE.
The cabinet itself was made using a variety of fabrication techniques, incorporating elements such as laser-cut acrylic panels, precision-cut pieces of wood, and printed boards with custom graphics.
Stop-A-Go was displayed at the 2024 MADE at Woodbury exhibition, where the public could interact with it. Participants at the installation had fun competing with each other for a high score.
