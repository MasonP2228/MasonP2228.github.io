HarvardX CS50

Over the summer and fall of 2025, I enrolled in and completed Harvard's CS50X: Web Programming with Python & JavaScript. This is an online course that teaches users how to create websites, with an emphasis on the use of Django. The course featured six different projects, the first five of which revolved around making mock versions of existing websites. The sixth project was open-ended, and served as the capstone project for the course. For my project, I decided to make a college roommate matching website, as I wanted to take advantage of Django's database capabilities.
You can watch a documentation video for the project here.
From the CS50 course, I improved my web development skills. I learned how to use Django, including its template capabilities and the Django ORM. I also learned how to make calls to an API using JavaScript, and how to make my own APIs in Django.
