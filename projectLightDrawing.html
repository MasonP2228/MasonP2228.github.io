Robotic Light Drawing

By using long exposure camera settings, it is possible to capture the path of a moving light in a single photograph. With this principle in mind, I created a couple of "light drawings" as a part of a small group project by mounting a light on Woodbury's KUKA robotic arm. The desired paths for the light drawings were created using the Grasshopper parametric design software before being exported as KRL code for the robot to follow. In addition to this, I programmed a Circuit Playground Express microcontroller to act as the light source, which allowed for multi-colored designs.
