CSGeo

For one of my finals in the Spring 2025 semester, I was asked to use p5.js, ml5.js, and a third JavaScript library of my choice to create an interactive web experience. I decided to make my own 3D-modeling program using the p5.csg library. I named my program CSGeo. It features 6 different built-in shapes, the ability to select, move, and delete shapes, and the ability to combine shapes into new ones. I also implemented an optional control scheme where users can move a cursor and click by using hand gestures in front of their web cameras.
In 2026, this project was selected for display at the Freeform Student Show on campus, where guests could interact with it.
