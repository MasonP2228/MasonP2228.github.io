Robotic Re-assembly Lab

For my final robotics project of the Spring 2025 semester, I learned more about robotic fabrication techniques. With a small group, I created a Grasshopper script for the university's KUKA robotic arm to pick up and place wooden blocks. First, we used the robot to place the blocks in a pillar-like configuration. Then, we had the robot disassemble the pillar and re-position the blocks in the shape of a chair. Once the robot had placed all of the blocks for our chair design, we nailed everything together.
From this experience, I learned a bit about using tools and I/O controls with the robotic arm. I also learned about the constraints and considerations that need to be kept in mind when using robots.
